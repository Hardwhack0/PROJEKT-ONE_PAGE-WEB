# PROJEKT-ONE_PAGE-WEB
## Název projektu
Guide na cviky
## Žák
Viktor Vlček
## Stručný popis tématu
Webová stránka bude fungovat jako moderní one-page prezentace zaměřená na fitness a posilování jednotlivých částí těla. Bude obsahovat interaktivní přehled segmentů (např. hrudník, záda, nohy, ramena, břicho), ke kterým se budou dynamicky načítat informace a doporučené cviky z JSON souborů. U každé partie bude krátký popis, výhody tréninku a přehled nejlepších cviků s obrázky nebo ikonami. Cílem je vytvořit přehledný, responzivní a moderní web, který pomůže začátečníkům i pokročilým orientovat se v tréninku.
## Cílová skupina 
Začátečníci i mírně pokročilí cvičenci, lidé hledající jednoduchý přehled cviků podle částí těla, uživatelé hledající vizuálně jasný a rychlý fitness návod.
## Návrh obsahu
1. O nás - krátké představení projektu, účel stránky, proč vznikla.
2. Partie - seznam segmentů (hrudník, záda, nohy, ramena, břicho, ruce).
3. Detaily partie - AJAXem načtené info o vybrané části těla + nejlepší cviky (popis, obrázek, návod na provedení).
4. Tréninková doporučení - základní pravidla, jak cviky kombinovat apod.
5. Kontakt - kontaktní údaje.
## Návrh designu
- Barevné ladění v energických, ale jemných tónech (modrá, šedá, oranžová).
- Blokové sekce pod sebou, každá partii reprezentuje ikona nebo silueta těla.
- Grid pro cviky (2–3 sloupce podle šířky obrazovky).
- Moderní prvky Tailwindu (karty, stíny, rounded rohy).
- Interaktivní hover efekty a animace při načítání cviků.
## Použité technologie
- Frontend: HTML,CSS,JavaScript,Tailwind
- Data: JSON soubory
- AJAX: dynamické načítání popisů a cviků
- Backend: jednoduchý PHP endpoint pro odeslání kontaktního formuláře
## Očekávaný přínos
- Práce s JSON a dynamickým obsahem
- AJAX a načítání dat bez reloadu
- Tvorba responzivního designu
